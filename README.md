# Validador de Credores – PCASP (Grupos 7 e 8)

Aplicação em Streamlit para validação automática de credores
entre Atos Potenciais Ativos e sua Execução, conforme PCASP.

## Uso
- Upload de arquivo CSV pelo painel
- Processamento temporário (nenhum arquivo salvo)
- Download do resultado em Excel

## Execução local
```bash
pip install -r requirements.txt
streamlit run app.py
